# Minigame Mania
Minigame Mania is a simple game based in the command prompt with implementation of 3 classic minigames, Hangman, Tic-Tac-Toe, and Trivia. Beat all 3 games and collect their prizes to win.

## Installation
Just download the v1.0 release ZIP file and run the MinigameMania.exe file to play!
